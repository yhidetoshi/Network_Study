- WAN(Wide Area Network)
  - 地理的に離れたLANとLANを相互に接続する広域ネットワーク
  - 電気通信事業者(サービスプロバイダまたはキャリア)が設置した機器や敷設したケーブルによって構築されたネットワークを利用して接続
  - 加入者側には専用の通信機器(DSLモデムはケーブルモデムが必要になる)
  
- WANで使用されるデータリンク層のカプセル化プロトコル
  - HDLC, PPP, フレームリレー, ATN
