- ACL
  - 特定のパケットフローを識別し制御するための条件文をまとめたリスト
  - トラフィックを分類することができる
  - 代表的な用途
    - **ネットワークアドレス変換(NAT)**
    - **ルートフィルタリング**
      - ルーティングプロトコルによるルーティングアップデートの際に、特定の経路情報を選択してアドバタイズできる
    - **ルートの再配布**
      - 経路情報を分類しながら再配布することができる
    - **VPN**
