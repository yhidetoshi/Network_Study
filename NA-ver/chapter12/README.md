- **DHCP**
  - 機能
    - IPアドレスなどの設定情報をクライアントに自動割り当て
    - DHCPサーバでプールされたアドレス範囲から、IPアドレスの割り当てと更新を行う
  - 設定可能な情報 
    - IPアドレス
    - サブネットマスク
    - リースの有効期限
    - デフォルトゲートウェイのIPアドレス
    - ドメイン名
    - DNSサーバのIPアドレス
    - NTPサーバのIPアドレス
